🎲 Duvido? ou Duvido? - Gerador de Charadas
Uma aplicação web interativa que gera charadas aleatórias consumindo uma API externa. O projeto apresenta uma interface moderna com efeitos de animação 3D e um design responsivo.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo.

Tailwind CSS: Estilização baseada em classes utilitárias, incluindo animações de perspectiva e rotação 3D.

JavaScript (ES6+): Manipulação de DOM e consumo de API assíncrona (fetch, async/await).

API de Charadas: Integração com https://api-charadas-eight.vercel.app para busca de conteúdo em tempo real.

✨ Funcionalidades
Busca Automática: Uma charada é carregada assim que a página é aberta.

Efeito Flip Card: O usuário clica no cartão para girá-lo em 180° e revelar a resposta com uma animação suave.

Nova Charada: Botão interativo que solicita uma nova pergunta à API e reseta a posição do cartão.

Tratamento de Erros: Mensagem amigável caso ocorra algum problema na conexão com o servidor.

🛠️ Como rodar o projeto
Clone este repositório ou baixe os arquivos.

Certifique-se de que os arquivos index.html e script.js estejam na mesma pasta.

Abra o arquivo index.html em qualquer navegador moderno.

Nota: É necessária uma conexão com a internet para carregar o Tailwind CSS via CDN e para o funcionamento da API.
